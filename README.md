# PAC-MAN NEON (パックマン ネオン)

ブラウザで動作する、ネオンサイバーパンク調のビジュアルと動的シンセ効果音を搭載したクラシックな「パックマン」ゲームです。

## 🎮 特徴

- **ネオンサイバーパンク・グラフィック**: CSSのグラスモーフィズムとCanvasの光彩（Glow）エフェクトを使用した、未来的でモダンなデザイン。
- **動的オーディオエンジン**: Web Audio APIを使用し、ゲーム開始のBGMやドットを食べる音（ワカワカ音）、ゴースト捕食音、死亡時の下降音、パワーエサの警告音などをプログラムでリアルタイム合成・再生。
- **本格的なゴーストAI**: 
  - **Blinky (赤)**: パックマンを直接追いかけます。
  - **Pinky (ピンク)**: パックマンの移動方向の4マス先を予測して先回りします。
  - **Inky (水色)**: Blinkyの位置とパックマンの先回り位置を結ぶベクトルから複雑な動きで挟み撃ちします。
  - **Clyde (オレンジ)**: パックマンに近いときは自分の縄張りへ逃げ、遠いときは追跡する気まぐれAI。
- **レスポンシブ & タッチコントロール**: デスクトップはもちろん、モバイル端末（スマートフォン）ではタッチ操作用の仮想十字コントローラーが自動表示されます。

## 🕹️ 操作方法

### PC (キーボード)
- **移動**: `W` / `A` / `S` / `D` または `↑` / `↓` / `←` / `→`
- **一時停止/再開**: `P`
- **ゲーム開始**: `Enter` または 「Start Game」ボタン

### モバイル (スマートフォン)
- 画面下部に表示される仮想コントローラー（矢印ボタン）をタッチして移動します。

## 🚀 ローカルでの実行方法

このプロジェクトは外部ライブラリを使用していない（Vanilla HTML/CSS/JS）ため、特別なビルドプロセスなしで実行できます。

1. このリポジトリ（またはフォルダ内のファイル）をローカルに準備します。
2. セキュリティ制約（ブラウザのESモジュールやWeb Audio APIの自動再生制限の回避など）に対応するため、簡易的なローカルサーバー経由での起動を推奨します。

#### VS Code を使用している場合:
- 拡張機能の「Live Server」をインストールし、`index.html`を右クリックして「Open with Live Server」を選択します。

#### Python がインストールされている場合:
フォルダ内で以下のコマンドを実行してサーバーを起動します。
```bash
python -m http.server 8000
```
起動後、ブラウザで `http://localhost:8000` にアクセスします。

## 🐙 GitHub への公開手順 (ユーザー様向け)

GitHub アカウント **`kenken6291`** に本プロジェクトを公開するための手順です。

### 1. ローカルでの Git コミット (エージェント側で実施済み)
すでにフォルダ内でローカル Git リポジトリの初期化と最初のコミットが作成されている状態にしています。

### 2. GitHub で新しいリポジトリを作成する
1. ブラウザで GitHub (https://github.com/) にログインします。
2. 右上の「+」アイコンをクリックし、**New repository** を選択します。
3. 以下のように設定します：
   - **Repository name**: `pacman-neon` (またはお好みの名前)
   - **Public/Private**: 公開したい場合は `Public` を選択します。
   - ※ `Initialize this repository with...` (READMEや.gitignoreの追加) は**すべてチェックを外したまま**にします（ローカルのファイルと競合するのを防ぐため）。
4. **Create repository** ボタンをクリックします。

### 3. ローカルリポジトリと GitHub を紐づけてプッシュする
GitHub にリポジトリを作成すると、以下のようなコマンドが表示されます。コマンドプロンプトや PowerShell 等で、本プロジェクトのディレクトリ（`C:\Users\user\.gemini\antigravity\scratch\pacman-game`）を開き、以下のコマンドを順番に実行してください。

```bash
# 1. リモートリポジトリ（GitHub）の登録
git remote add origin https://github.com/kenken6291/pacman-neon.git

# 2. メインブランチの名前を 'main' に変更
git branch -M main

# 3. GitHubへコードをプッシュ
git push -u origin main
```

これで GitHub へゲームが公開されます！

### 💡 GitHub Pages でゲームを無料公開する
GitHubにプッシュした後、誰でも遊べるようにウェブ上にホスティングすることができます：
1. 作成した GitHub リポジトリの **Settings** タブを開きます。
2. 左側のメニューから **Pages** を選択します。
3. **Build and deployment** の `Source` を `Deploy from a branch` に設定します。
4. `Branch` で `main` を選択し、フォルダは `/ (root)` のまま **Save** をクリックします。
5. 数分待つと、画面上部に公開URL（例：`https://kenken6291.github.io/pacman-neon/`）が表示され、ブラウザからアクセスしてプレイできるようになります。
